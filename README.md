# Installation Guide

## Clone
1. ```git clone https://github.com/DanielaMazur/FlightTicketsBookingSystem.git```
2. ```cd FlightTicketsBookingSystem/gateway```

## Setup the Gateway (Windows)
1. ```py -3 -m venv venv```
2. ```venv\Scripts\activate```

## Run the Gateway
1. ```pip install -r requirements.txt```
2. ```flask --app app run ```
